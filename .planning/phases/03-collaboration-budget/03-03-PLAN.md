| Boundary                          | Description                              |

| Collaborator input -> budget APIs | Untrusted numeric/currency values        |

| API -> database                   | Trip scope and role-based write controls |

| Threat ID  | Category               | Component                | Disposition | Mitigation Plan                                                                  |

| T-03-03-02 | Repudiation            | Expense edits            | mitigate    | Store updated_by and updated_at fields for each budget/expense mutation.         |
